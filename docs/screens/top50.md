# /top50（Top50）

## この画面の目的

- 候補銘柄を「一覧で素早く比較」するための画面です。
- 毎日最初に見る入口です。

## できること

- 日米の候補銘柄をランキング表示
- 市場/信頼度/シグナル有無で絞り込み
- 各銘柄の詳細画面に移動

## 項目の意味（初心者向け）

- `Rank`
  - 順位です。1位ほど総合評価が高い候補です。
- `Score / Edge`
  - 総合点と優位性の点数です。高いほど有利な可能性があります。
- `Signal`
  - 現在の条件で「採用対象かどうか」です。
- `欠損 / 流動性`
  - 欠損: データ不足の割合。高いと判断が不安定になります。
  - 流動性: 売買しやすさ。低いと約定しにくくなります。
- `Confidence`
  - 推定の信頼度です（High/Medium/Low）。

## フィルタの使い方

- `市場`
  - `JP` または `US` だけに絞れます。
- `Confidence`
  - High/Medium/Low で絞り込みます。
- `Signalのみ`
  - `Yes` にすると、実行候補だけに絞れます。
- `Sort`
  - `Rank` または `Score` で並び替えます。

## 判断の目安（運用しやすい順）

- まず `Signalのみ = Yes` で候補を減らす
- 次に `欠損率が低い` + `流動性OK` を優先
- 最後に `Score / Edge` が高い順で詳細確認

## 毎日の確認手順（3分）

1. `/top50` を開く
2. `Signalのみ = Yes` にする
3. 上位3〜5銘柄を詳細画面へ開く
4. `/edge` と `/execution` で実行可否を最終確認

## よくあるつまずき

- 候補が多すぎる
  - まず `Signalのみ = Yes` にしてから見てください。
- 候補が出ない
  - `Confidence` フィルタを `ALL` に戻してください。
