# /reports/[securityId]（銘柄詳細）

## この画面の目的

- 1銘柄を深掘りして、採用/見送りを判断する画面です。

## できること

- スコアや時系列の確認
- 採用理由/見送り理由の確認
- 関連する評価情報の確認

## URL例

- `JP:7203` の場合: `/reports/JP%3A7203`
- `US:AAPL` の場合: `/reports/US%3AAAPL`

## 主な確認ポイント

- 直近で評価が改善しているか
- 欠損や流動性に問題がないか
- 採用理由が一時要因だけでないか

## 使い方（推奨）

1. `/top50` から気になる銘柄を開く
2. 理由と時系列を確認
3. `/edge` と `/execution` で最終確認
